## 頁面 1: 啟動畫面 (Splash Screen)

### 目的
品牌識別、載入必要資源、檢查登入狀態

### 佈局結構 (ASCII)
```
╔═══════════════════════════════╗
║                               ║
║        [漸變背景]              ║
║     深紫 → 紫紅漸層            ║
║                               ║
║          ⚔️ 🐉                ║
║      [Logo 圖示]              ║
║                               ║
║    POMODORO DUNGEON           ║
║      [App 名稱]               ║
║                               ║
║    「專注學習的冒險旅程」      ║
║      [Slogan]                 ║
║                               ║
║                               ║
║        ●●●○○                  ║
║    [載入動畫點]               ║
║                               ║
║     v1.0.0                    ║
║   [版本號 - 底部]             ║
╚═══════════════════════════════╝
```

### 視覺元素

**背景**
- 漸變: `linear-gradient(180deg, #1E1B4B 0%, #7C3AED 50%, #EC4899 100%)`
- 微妙的星點閃爍動畫 (CSS animation)

**Logo 區域**
- 主圖示: ⚔️ + 🐉 組合 (96px × 96px)
- 圖示外圈: 紫色光暈脈動效果
- 文字: "POMODORO DUNGEON" (36px, Bold, White)
- Slogan: 「專注學習的冒險旅程」(14px, Purple-200)

**載入指示器**
- 5個小圓點水平排列
- 依序亮起動畫 (白色 → 紫色)
- 循環播放

### 動畫效果

1. **進場** (0-1s):
   - Logo 從上方淡入 + 輕微彈跳
   - 光暈由小變大

2. **載入中** (1-3s):
   - 點狀載入器動畫
   - 背景星點閃爍

3. **退場** (3s):
   - 整體淡出
   - 轉場至登入頁或主頁

### 狀態處理

- **首次安裝**: 導向 Onboarding
- **未登入**: 導向登入頁
- **已登入**: 導向主頁
- **載入失敗**: 顯示錯誤訊息

### 互動流程

#### 開始計時流程
1. 用戶點擊「開始」按鈕
2. 按鈕文字變為「暫停」
3. 計時器數字開始倒數
4. 進度條開始填充
5. 勇者開始行走動畫
6. 怪物開始脈動動畫
7. 每秒更新一次所有狀態

#### 暫停流程
1. 用戶點擊「暫停」按鈕
2. 所有動畫停止
3. 按鈕文字變為「繼續」
4. Toast 提示: "已暫停"

#### 完成流程
1. 計時器歸零
2. 所有動畫停止
3. 觸發完成動畫:
   - 怪物消失 (淡出)
   - 金幣從怪物位置掉落
   - 勇者勝利跳躍
   - 全螢幕慶祝效果
4. 顯示獲得獎勵 Modal:
```
   ┌─────────────────────┐
   │       勝利！        │
   │                     │
   │        💰          │
   │     [金幣動畫]     │
   │                     │
   │    獲得 10 金幣     │
   │                     │
   │   ⚡ +25 XP        │
   │   🔥 連勝 +1        │
   │                     │
   │   [繼續] [休息]    │
   └─────────────────────┘
```
5. 更新統計數據 (數字彈跳)
6. 自動切換到休息模式 (可選)
