# 🐉 Pomodoro Dungeon - 地牢專注

> 將番茄鐘學習化身為史詩冒險旅程

一款結合遊戲化學習的番茄鐘應用程式,專為學生族群設計。通過地牢冒險的遊戲化機制,讓專注學習變得更有動力與樂趣。

## ✨ 核心特色

### 🎮 遊戲化學習
- **地牢冒險主題**: 化身勇者,擊敗分心怪物
- **勇者行走動畫**: 專注時勇者向怪物前進,視覺化你的進度
- **等級系統**: 持續學習升級角色,解鎖新稱號
- **成就徽章**: 收集各種學習成就,展示你的努力

### ⏱️ 番茄鐘核心
- **25 分鐘專注 + 5 分鐘休息** 經典番茄鐘設定
- **實時動畫反饋**: 勇者行走、怪物呼吸、進度條填充
- **完成慶祝**: 擊敗怪物時的震撼動畫與金幣獲得
- **彈性設定**: 可自訂專注/休息時長

### 💰 獎勵系統
- **虛擬貨幣**: 完成專注獲得金幣
- **真實獎勵**: 兌換校園優惠券、會員、禮品
- **獎勵商城**: 實體、數位、體驗三大類獎勵
- **即時兌換**: 郵件發送兌換碼

### 🏆 社交競爭
- **學校排行榜**: 與同校學生競爭
- **公會系統**: 組隊為學校榮譽而戰
- **個人排名**: 追蹤自己的進步軌跡
- **賽季獎勵**: 定期重置,前三名獲得獎勵

### 📊 數據追蹤
- **詳細統計**: 今日/本週/本月/總計數據
- **趨勢圖表**: 視覺化你的學習曲線
- **時段分析**: 了解你最高效的學習時段
- **連勝記錄**: 激勵持續學習習慣

## 🎨 設計系統

### 色彩配置
```css
/* 主色調 - 地牢深紫 */
--primary: #8B5CF6
--primary-dark: #7C3AED
--primary-light: #A78BFA

/* 次要色 - 神秘粉 */
--secondary: #EC4899
--secondary-dark: #DB2777

/* 專注模式 - 火焰橙紅 */
--focus: linear-gradient(135deg, #DC2626, #EA580C)

/* 休息模式 - 恢復綠 */
--break: linear-gradient(135deg, #059669, #10B981)

/* 獎勵金幣 - 寶藏金 */
--gold: #F59E0B
```

### 設計原則
1. **移動端優先**: 針對手機使用優化
2. **暗色主題**: 減少眼睛疲勞,營造地牢氛圍
3. **大按鈕**: 最小 48×48px 觸控友善
4. **即時反饋**: 每個操作都有視覺動畫
5. **一致性**: 統一的設計語言

## 📖 使用指南

### 如何開始專注?
1. 進入主頁 (底部導航 🎯 專注)
2. 選擇「專注模式」或「休息時間」
3. 點擊「▶️ 開始」按鈕
4. 勇者開始向怪物前進!
5. 25 分鐘後擊敗怪物,獲得金幣 💰

### 如何兌換獎勵?
1. 進入商城 (底部導航 🛒 商店)
2. 瀏覽或篩選獎勵類別
3. 點擊心儀獎勵的「購買」按鈕
4. 確認兌換資訊
5. 兌換碼將發送到你的 Email

### 如何查看排名?
1. 進入排行榜 (底部導航 🏆 排名)
2. 切換「公會排名」或「個人排名」
3. 選擇時間範圍 (本週/本月/總排行)
4. 查看你的排名與進步空間

## 🎯 核心功能詳解

### 勇者行走動畫 ⭐
這是應用的視覺核心特色,當你開始專注時:
- 勇者從左側起點出發
- 以與計時器同步的速度向右移動
- 上下彈跳 (模擬行走)
- 左右搖擺 (增加生動感)
- 腳下塵土效果 💨
- 25 分鐘後到達怪物位置並擊敗它!

**技術實現**: 使用 Motion 的多層動畫組合
```tsx
<motion.div
  animate={{
    x: isActive ? ['0%', '85%'] : '0%',
    y: [0, -8, 0],
    rotate: [-5, 5, -5],
  }}
  transition={{
    x: { duration: 1500, ease: 'linear' },
    y: { duration: 0.6, repeat: Infinity },
    rotate: { duration: 0.6, repeat: Infinity },
  }}
>
  🚶
</motion.div>
```

### 完成慶祝動畫
精心編排的 5 步動畫序列:
1. **勝利 (0-500ms)**: 勇者到達怪物位置
2. **消失 (500-1000ms)**: 怪物淡出
3. **掉落 (1000-1500ms)**: 金幣從怪物處掉落
4. **飛行 (1500-2000ms)**: 金幣飛向頂部餘額
5. **更新 (2000-2500ms)**: 數字彈跳更新 +10

## 🎓 設計決策

### 為什麼選擇地牢主題?
1. **具象化抽象概念**: 學習 = 冒險,分心 = 怪物
2. **吸引目標用戶**: 大學生熟悉遊戲文化
3. **視覺一致性**: 暗色調符合專注氛圍
4. **豐富的視覺元素**: 勇者/怪物/寶箱/金幣

### 為什麼勇者要行走?
1. **視覺化進度**: 比單純進度條更有趣
2. **建立連結**: 用戶與勇者共同前進
3. **增加陪伴感**: 不是孤單的學習
4. **強化隱喻**: 學習是一場旅程

### 為什麼提供真實獎勵?
1. **外在激勵**: 補充內在學習動機
2. **即時滿足**: 看得見的成果
3. **校園整合**: 與學生生活結合
4. **差異化**: 區別於其他番茄鐘 App
